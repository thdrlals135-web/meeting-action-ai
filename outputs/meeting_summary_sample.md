# 노바드림 캠페인 회의록 요약

## 1. 회의 개요

- 회의명: 노바드림 다음달 캠페인 제안 사전 정렬 회의
- 참석자:
  - 지훈: 마케팅 팀장
  - 수아: 퍼포먼스 마케터
  - 채린: 콘텐츠 디자이너
- 주요 목적:
  - 다음달 캠페인 제안 전 채널 전략, 성과 데이터, 크리에이티브 방향, 광고주 컨펌 사항을 정리한다.

## 2. 주요 논의사항

### 2.1 성과 데이터 및 픽셀 이슈

지난 캠페인 전환 수치에서 Meta와 GA 데이터가 맞지 않는 문제가 확인되었다. 수아는 Pixel 이벤트가 중복 발화되는 것이 원인으로 보인다고 언급했으며, 보정 후 데이터를 다시 산출하기로 했다.

이 이슈는 제안서의 성과 근거와 직접 연결되므로 우선순위가 높은 작업으로 분류된다.

### 2.2 채널 운영 전략

초기 캠페인 채널은 Meta와 Instagram 중심으로 운영하고, YouTube는 2주차 옵션으로 제안서에 포함하는 방향으로 정리되었다. YouTube는 영상 소재 리드타임이 길어 1주차부터 집행하기에는 부담이 있다는 의견이 있었다.

### 2.3 비주얼 및 카피 수정

채린은 비주얼 카드 순서와 빈 슬롯 카피를 내일 오전까지 1차 정리하기로 했다. 또한 메인 헤드라인, 랜딩 첫 화면, CTA 버튼 카피는 우선 수정이 필요한 영역으로 논의되었다.

다만 기존 A/B 테스트가 진행 중이므로, 카피 변경 시 기존 테스트를 종료하고 변경된 카피 기준으로 다시 세팅해야 한다.

### 2.4 신제품 누끼 컷 미수령

광고주 측 신제품 누끼 컷이 아직 도착하지 않았으며, 이 자료가 없으면 메인 비주얼 시안 작업이 지연될 수 있다. 지훈은 광고주 담당자에게 오늘 안으로 재요청하기로 했다.

## 3. 결정사항

- 캠페인 1주차는 Meta와 Instagram 중심으로 제안한다.
- YouTube는 2주차 옵션으로 제안서에 포함한다.
- 픽셀 이벤트 중복 발화 문제를 보정한 뒤 성과 데이터를 다시 산출한다.
- 메인 헤드라인, 랜딩 첫 화면, CTA 버튼 카피를 우선 수정한다.
- 광고주 컨펌은 지훈이 담당한다.
- 신제품 누끼 컷이 오늘 안에 오지 않으면 임시 컷 사용을 고려한다.

## 4. 리스크 및 확인 필요 사항

- Meta와 GA 전환 수치 불일치가 제안서 신뢰도에 영향을 줄 수 있다.
- 신제품 누끼 컷 미수령 시 비주얼 시안 작업이 지연될 수 있다.
- CTA 변경 시 전환 추적 이벤트를 다시 확인해야 한다.
- 기존 A/B 테스트가 중단되므로 광고주 보고 시 비교 기간 단절을 설명해야 한다.
